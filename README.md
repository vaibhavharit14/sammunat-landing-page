# Sammunat Landing Page

A modern, responsive landing page built with **Next.js (App Router)** and **TailwindCSS v4**.  
This project was created as part of the internship assignment for **Sammunat (USA)**.

Vercel :- https://sammunat-landing-page-nu.vercel.app/

##  Features
- Responsive design (mobile, tablet, desktop)
- Clean UI with gradient backgrounds and reusable components
- Sections:
  - Hero (logo, heading, CTA buttons)
  - About (short intro + highlights)
  - Features (cards with icons)
  - Contact (form with validation)
  - Footer (links + copyright)
- Built with **Next.js 16** and **TailwindCSS v4**
- Easy deployment on **Vercel/Netlify**

##  Tech Stack
- **Framework:** Next.js (React + App Router)
- **Styling:** TailwindCSS v4
- **Language:** TypeScript
- **Hosting:** Vercel (recommended)
